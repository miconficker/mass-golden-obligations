## Mass Golden Obligations

Simple QOL mod to cash in all those favors. Creates a decision to demand payment for hooks using Golden Obligations perk.

Features:
3 options to filter if receipient has enough gold.
1. Any payment
2. Half + payment
3. Full payment only

Update:
-reworked to reduce the notification spam from demanding hundreds of hooks.
